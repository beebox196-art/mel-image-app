# Mel's Imagen Studio

A simple image generation app powered by Google Imagen, built for Mel by Bee.

## What This Is

Mel's personal image studio - type what you want to see, click generate, get an image. Uses Adam's $300 Google Cloud credit.

## How to Use

1. Open the app URL
2. Type a description of what you want
3. Click "Generate Image"
4. Download your creation

## Tips for Better Images

- Be specific with details
- Include style (photo, painting, cartoon, digital art)
- Add lighting (sunset, dramatic, soft, studio)
- Mention mood (peaceful, energetic, mysterious)
- Describe composition (close-up, wide shot, portrait)

## Examples

- "A serene mountain lake at sunset, photorealistic, warm golden lighting, peaceful atmosphere"
- "A futuristic cyberpunk city street at night, neon lights reflecting in rain puddles, cinematic"
- "A whimsical treehouse in a magical forest, Studio Ghibli style, soft morning light"

## Technical Details

- **Model:** Google Imagen 3
- **Framework:** Streamlit
- **Backend:** Google Vertex AI
